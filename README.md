# NovaCart Platform

NovaCart Platform is an enterprise-grade cloud-native e-commerce platform built to simulate real-world software engineering, Agile, DevSecOps, and cloud infrastructure practices.

## Project Goals

- Build a scalable e-commerce platform.
- Apply Agile Scrum practices using Jira.
- Use a monorepo architecture.
- Implement frontend and backend services.
- Run PostgreSQL and DynamoDB locally.
- Add automated testing and code-quality checks.
- Deploy the platform to AWS using modern DevOps practices.
